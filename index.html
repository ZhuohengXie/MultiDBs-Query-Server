Multi-database system query with PrestoDB Server

Desciption of Presto DB Project

Hi everyone! Presto is an open source distributed SQL query engine for running interactive analytic queries against data sources of all sizes ranging from gigabytes to petabytes. Our group use Presto DB to allow users do SQL query from MetaStore Project's database and allow user to save query results in indicated database.

1. In our web interface, we provide databases with their metadata(Database type, Database name......) to help user write SQL query.

2. After execute the query, the user could save the returned result into indicated database by click save.
